# Ardujimmy-Bicopter-with-NRF24L01-PPM
This is ver1.0 ardujimmy Bicopter project with arduino Nano 328p NRF24L01 as TX RX in PPM signal
<p>Selamat datang di project Bicopter berbasis arduino Ardujimmy. Kita akan menggunakan NRF24L01, PPM Receiver dan Multiwii 2.3 Firmware pada project ini.</p>
